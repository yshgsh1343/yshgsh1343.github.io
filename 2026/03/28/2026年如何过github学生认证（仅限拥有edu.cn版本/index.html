前言

由于近两年github学生认证逐渐收紧到真学生都过不去认证，且国内直连 GitHub 本就困难，而审核要求关闭 VPN 并共享地理位置。用校园网网页刷不出来用VPN则IP 地址对不上。于是即使是真学生也需要采取特殊手段，下面是个人通过截图证明。

准备工作

  1. edu邮箱

  2. 可以使用加密DNS服务的浏览器，不推荐Edge。

  3. 一根数据线与Irium Webcam，手机需要前往Play商店下载。电脑端https://iriun.com/

注意事项

  1. 全程避免任何VPN工具使用，不要在申请过程中反复开关 VPN

  2. Github实名较为关键不可省略。痛失网名

步骤

1.修改DoH

修改DoH使得可以在国内环境免VPN工具直连Github。笔者此处使用为Brave浏览器，设置-隐私与安全-高级，打开使用安全DNS把DNS提供商改为Cloudfare（1.1.1.1）。

Chorme浏览器位置

Tips：如果 Chrome 提示“浏览器由贵单位管理”导致无法修改，请：

  1. Win+R 输入 RegEdit 运行注册表。

  2. 删除 \HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Google\Chrome 整个目录。

    2. 修改个人信息 （Profile易忽略）

    2.1 修改姓名

下面的name一定要改真实姓名。

2.2 账单信息：

3.绑定edu.cn邮箱与开启2FA

笔者注册github就用的edu.cn所以此处不赘述，仅贴链接https://github.com/settings/emails

2FA验证

安卓在软件商城下载Authenticator扫码即可，记得保存recovery code。

4.开始申请学生认证

可以看见如果第二步绑定成功这一步可以直接选择select this school不需要填写详细信息。

理论上这一步share location就可以通过但可以加一道保险。

打开开发者工具（依旧Brave举例子

位置改为其他，经纬度使用http://jingweidu.757dy.com查询。越精确越好。

Chorme

5.按要求上传个人信息

选择8.other，先用win+x打开设备管理器禁用笔记本自带摄像头，数据线连接手机与电脑（同一wifi也行开热点），两台设备打开Irium Webcam准备拍屏幕。

  • 方法一：记事本大法

注意Study Form一栏填写Distance learning ，下边那句Student xxx,is allowed to study via distance learning别省略。这样如果后续爆出询问为何不在当前地址就可以选择distance learning。

  • 方法二：学信网申请学籍报告下载，丢到google翻译成英文。再采用任意pdf修改器修改字体大小（学信网那个报告字太小了github识别容易失误）+改为Distance learning

手机拍摄电脑屏幕，如果后续跳出第六步为何不在对应位置选择distance learning提交。
